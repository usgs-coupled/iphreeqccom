RunString Method

Runs phreeqc using the given string as input. Returns the number of errors encountered during the run.

expression.RunString(Input )

expression    Required. An expression that returns a phreeqc object.

Input   Required. The string containing phreeqc input.

Remarks

The RunString method cannot be called until a database has been successfully loaded by one of the LoadDatabase methods: LoadDatabase or LoadDatabaseString.

Example

This example stores phreeqc input into a string and runs it through the phreeqc simulator.


  JScript

var Phreeqc = new ActiveXObject("PhreeqcCOM.Phreeqc");
Phreeqc.LoadDatabase("phreeqc.dat");

var sInput = "SOLUTION 1\n";
sInput += "temp 25.0\n";
sInput += "pH   7.0\n";

Phreeqc.OutputOn = true;
Phreeqc.RunString(sInput);


  VBScript

Dim Phreeqc, sInput
Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")

sInput = "SOLUTION 1" & vbCrLf
sInput = sInput & "temp 25.0" & vbCrLf
sInput = sInput & "pH   7.0" & vbCrLf

Phreeqc.OutputOn = True
Phreeqc.RunString(sInput)
