RowCount Property

Returns the number of rows in the selected output array. Read-only Long.

Example

This example runs "ex2" and displays the number of columns contained within the selected output array.


  JScript

var Phreeqc = new ActiveXObject("PhreeqcCOM.Phreeqc");
Phreeqc.LoadDatabase("phreeqc.dat");
Phreeqc.RunFile("ex2");
var shell = new ActiveXObject("wscript.shell")
shell.Popup("The current selected output array contains " +
    Phreeqc.RowCount + " rows.");


  VBScript

Dim Phreeqc
Set Phreeqc = CreateObject("PhreeqcCOM.Phreeqc")
Phreeqc.LoadDatabase("phreeqc.dat")
Phreeqc.RunFile("ex2")
MsgBox "The current selected output array contains " & _
    Phreeqc.RowCount & " rows."
