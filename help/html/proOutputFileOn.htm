OutputFileOn Property

True to turn on output to the phreeqc.out  file. The initial setting is False. Read/write Boolean.

Remarks

The OutputFileOn property is used only by the Run methods: RunAccumulated, RunFile and RunString.

Example

This example runs ex2  with the phreeqc.out  file turned on.


  JScript

var iphreeqc = new ActiveXObject("IPhreeqcCOM.Object");
iphreeqc.LoadDatabase("phreeqc.dat");
iphreeqc.OutputFileOn = true;
iphreeqc.RunFile("ex2");
WScript.Echo("see phreeqc.out");

  VBScript (Deprecated)

Dim iphreeqc
Set iphreeqc = CreateObject("IPhreeqcCOM.Object")
iphreeqc.LoadDatabase("phreeqc.dat")
iphreeqc.OutputFileOn = True
iphreeqc.RunFile("ex2")
WScript.Echo("see phreeqc.out")
